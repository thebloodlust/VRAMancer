🚀 POUR INSTALLER SUR VOTRE MAC
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ÉTAPE 1 : Transférer les fichiers
──────────────────────────────────
Copiez sur votre Mac :
• VRAMancer-1.0.0-macOS.tar.gz
• create_dmg_on_macos.sh

ÉTAPE 2 : Créer le .dmg
────────────────────────
Ouvrez un Terminal sur votre Mac :

  cd ~/Downloads  # (ou dossier des fichiers)
  chmod +x create_dmg_on_macos.sh
  ./create_dmg_on_macos.sh

→ Crée : VRAMancer-1.0.0-macOS.dmg

ÉTAPE 3 : Installer
────────────────────
• Double-cliquez sur VRAMancer-1.0.0-macOS.dmg
• Glissez VRAMancer.app dans Applications
• Lancez depuis Launchpad

ÉTAPE 4 : Premier lancement
────────────────────────────
• Double-clic sur VRAMancer dans Applications
• Si "App non vérifiée" : clic droit → Ouvrir
• Installation auto des dépendances (2-5 min)
• API démarre sur http://localhost:5030
• Interface s'ouvre automatiquement
