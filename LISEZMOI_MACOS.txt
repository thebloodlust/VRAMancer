═══════════════════════════════════════════════════════════════════════════
  SOLUTION 1 : DEPUIS LE DOSSIER CLONÉ (VRAMancer-main)
═══════════════════════════════════════════════════════════════════════════

Vous êtes dans : /Users/JEJE/Downloads/VRAMancer-main

Exécutez ces commandes :

# 1. Nettoyer
rm -rf build_macos dmg

# 2. Télécharger le script FIXÉ depuis GitHub
curl -L -O https://github.com/thebloodlust/VRAMancer/raw/main/create_dmg_on_macos.sh

# 3. Rendre exécutable
chmod +x create_dmg_on_macos.sh

# 4. Créer le DMG
./create_dmg_on_macos.sh

# 5. Ouvrir
open VRAMancer-1.1.0-macOS.dmg

═══════════════════════════════════════════════════════════════════════════
  INSTALLATION
═══════════════════════════════════════════════════════════════════════════

1. Le DMG s'ouvre automatiquement
2. Glisser VRAMancer.app dans le dossier Applications
3. Lancer : open /Applications/VRAMancer.app


═══════════════════════════════════════════════════════════════════════════
  SI MACOS BLOQUE L'APP (Gatekeeper)
═══════════════════════════════════════════════════════════════════════════

# Autoriser l'app
xattr -d com.apple.quarantine /Applications/VRAMancer.app

# Relancer
open /Applications/VRAMancer.app
