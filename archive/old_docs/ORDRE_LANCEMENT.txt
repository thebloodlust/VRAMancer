                            ÉTAPE 1 - API (OBLIGATOIRE)
===============================================================================

🚀 LANCEMENT API PERMANENTE:

   Option A - Automatique:
   vramancer_auto.bat
   → Lance l'API automatiquement et propose les interfaces

   Option B - Manuel:
   api_permanente.bat
   → Gardez cette fenêtre ouverte en permanence

   Option C - Via menu:
   vramancer_menu_simple.bat → Option 1

VÉRIFICATION:
✅ Une fenêtre reste ouverte avec "API VRAMancer sur port 5030"
✅ Le script check_api.py confirme l'API active

===============================================================================
                               PROBLÈMES COURANTS
===============================================================================

❌ ERREUR: "API VRAMancer NON ACCESSIBLE"
   SOLUTION: Lancez d'abord api_permanente.bat

❌ ERREUR: Interface reste "en chargement"
   SOLUTION: Vérifiez que l'API fonctionne (fenêtre ouverte)

❌ ERREUR: "Port 5030 occupé"
   SOLUTION: Tuez les processus Python puis relancez l'API

❌ ERREUR: Interface Qt/Tkinter ne s'ouvre pas
   SOLUTION: Utilisez les interfaces web (10, 12, 13)

===============================================================================
                               MÉTHODES DE LANCEMENT
===============================================================================

🚀 MÉTHODE 1 - TOUT AUTOMATIQUE (RECOMMANDÉE):
   vramancer_auto.bat
   → Vérifie et lance l'API, puis propose les interfaces

🚀 MÉTHODE 2 - MENU PRINCIPAL:
   vramancer_menu_simple.bat

   1. api_permanente.bat (gardez ouvert)
   2. python dashboard\dashboard_qt.py (par exemple)

   python check_api.py
