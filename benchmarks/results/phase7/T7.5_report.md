# T7.5 — Test éclair : encodage différentiel temporel des activations de frontière

Branch: `phase7/T7.5-delta-probe`. Script jetable :
`benchmarks/probe_delta.py` (1 journée max, test de faisabilité — pas une
implémentation).

## Hypothèse

Les états cachés à la frontière inter-GPU évoluent peu d'un token de décodage
au suivant ; transmettre le **delta** quantifié coûterait moins que l'état
complet.

## Méthode

- `Qwen/Qwen2.5-7B-Instruct`, chargé avec `device_map="auto"` sur 2 GPU
  (`accelerate`). `hf_device_map` confirme la frontière : couches 0-11 sur
  `cuda:0`, couche 12+ sur `cuda:1` → frontière = entrée de
  `model.model.layers[12]`.
- Hook `register_forward_pre_hook` sur `layers[12]` capturant
  `hidden_states` en entrée à chaque appel forward.
- Génération greedy (`do_sample=False`, `use_cache=True`) de 100 tokens sur
  le prompt fixe `"Write a Python function that parses a CSV file and returns
  a dict."`.
- 1 capture par step de décodage (`seq_len==1`) → 99 captures → 98 valeurs
  `r_t = norm(h_t - h_{t-1}) / norm(h_t)` pour t=2..100.

## Résultat

| min | médiane | max |
|---|---|---|
| 0.5544 | 0.8789 | 1.2511 |

Seuil de décision : médiane < 0.3 → hypothèse vivante ; médiane ≥ 0.3 →
fermeture définitive.

**Médiane mesurée = 0.879**, soit ~3× le seuil de 0.3, et même le **minimum**
observé (0.554) reste au-dessus du seuil. L'état caché à la frontière change
de **~88% de sa propre norme** d'un token au suivant, en moyenne — ce n'est
pas un signal "lentement variable" exploitable par un encodage différentiel
simple.

## Verdict

**FERMÉ DÉFINITIVEMENT** (résultat négatif documenté, valide per règle Phase 7
#4).

- L'hypothèse d'un delta temporel exploitable entre activations de frontière
  consécutives est rejetée pour Qwen2.5-7B sur ce point de coupure
  (couche 12/28, frontière `cuda:0`→`cuda:1`).
- Pas de tâche de suivi à ouvrir pour cette piste.
- Hypothèse alternative (hors scope T7.5, pour mémoire) : la variance élevée
  pourrait venir de la nature résiduelle de l'architecture transformer (les
  blocs résiduels ajoutent une contribution de norme comparable à chaque
  couche par construction) — un encodage différentiel **spatial** (entre
  couches plutôt qu'entre steps temporels) ou une compression de la
  contribution résiduelle elle-même pourraient être des pistes plus
  prometteuses, mais ce n'est pas ce que T7.5 testait.
