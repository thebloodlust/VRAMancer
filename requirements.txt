# Single source of truth for dependencies: pyproject.toml
# ([project.dependencies] + [project.optional-dependencies]).
#
# This file exists only for tooling (Docker, CI, Makefile) that expects
# a requirements.txt. It installs VRAMancer itself with the "gpu" extra
# (torch/transformers/accelerate/...), which matches the previous
# "lite/base server" profile.
#
# For development, prefer an editable install with extras:
#   pip install -e .            -> CLI / orchestrator only, no torch
#   pip install -e .[gpu]       -> + torch/transformers/accelerate
#   pip install -e .[full]      -> everything (GUI, quantization, GGUF, ...)
#   pip install -e .[windows]   -> Windows dashboards profile
